# WinForms Product Pricing App

A simple and educational **Windows Forms** application that allows users to add products, calculate total prices, and list them with a clear formatted output.  
This project is built for learning purposes and demonstrates essential WinForms components and fundamental C# concepts.

---

##  Features

- Add products with:
  - Product Name  
  - Quantity  
  - Unit Price  
- Automatic total price calculation  
- Formatted output inside a ListBox  
- Clean and simple struct-based model  
- Overridden `ToString()` method for readable listings  
- Beginner-friendly code structure  

---

## Technologies Used

- C#  
- .NET 10 (Windows Forms)  
- Visual Studio 2026 
- Structs  
- Object-Oriented Programming basics
